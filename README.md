# GenTiny

URL shortener

<h3> Execution: </h3>

* apt-get install python3

* apt-get install git

* git clone https://github.com/AngelSecurityTeam/GenTiny

* cd GenTiny

* pip3 install requests

* python3 gentiny.py o python3 gentiny2.py
